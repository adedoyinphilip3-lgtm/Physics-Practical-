Spiral Spring Oscillation Experiment

Question

Suspend the spiral spring vertically using a retort stand. Attach a 40 g mass hanger to the spring and pull it slightly downward. Release it so that it oscillates vertically. Measure the time taken for 20 oscillations. Calculate the period (T) and T². Repeat for additional masses of 20 g, 40 g, 60 g and 80 g.

Observation Table

Mass (g) Time for 20 Oscillations (s) Period T (s) T² (s²)
40 16.0 0.80 0.64
60 18.0 0.90 0.81
80 20.0 1.00 1.00
100 22.0 1.10 1.21
120 24.0 1.20 1.44

Sample Calculation

For 40 g:

T = 16.0 ÷ 20 = 0.80 s

T² = (0.80)² = 0.64 s²

Graph

  • X-axis: Mass (g)
  • Y-axis: T² (s²)
  • Plot all points neatly.
  • Draw the best-fit straight line.

Precautions

  • Release the mass gently.
  • Ensure the spring oscillates vertically.
  • Avoid parallax error.
  • Use the same reference point when counting oscillations.

Sources of Error

  • Reaction time while using the stopwatch.
  • Air resistance.
  • Internal friction in the spring.
